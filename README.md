# Qwen / Wan Image Generation - ComfyUI 自定义节点

基于阿里云百炼平台 DashScope API 的 ComfyUI 图像生成节点，包含两个独立节点：

- **Qwen Image Generation** — 通义万相 qwen-image-2.0 系列
- **Wan 2.6 Image Generation** — 万相 wan2.6-image（图像编辑 + 图文混排）

## 依赖

```
dashscope>=1.25.8
requests>=2.28.0
Pillow>=9.0.0
numpy>=1.23.0
torch>=2.0.0
```

> `torch`、`numpy`、`Pillow`、`requests` 通常已随 ComfyUI 安装，一般只需额外安装 `dashscope`。

### 快速安装

```bash
pip install dashscope
```

## 配置

使用前需设置阿里云百炼平台的 API Key 环境变量：

```bash
export DASHSCOPE_API_KEY="sk-xxxxxxxxxxxxxxxxxxxxxxxx"
```

API Key 获取地址：<https://help.aliyun.com/zh/model-studio/get-api-key>

## 节点 1：Qwen Image Generation

路径：`Qwen` → `Qwen Image Generation`

### 输入参数

| 参数 | 类型 | 说明 |
|---|---|---|
| model | 下拉选择 | `qwen-image-2.0`（默认）/ `qwen-image-2.0-pro` |
| text | 文本框 | 图像生成提示词，可为空 |
| n | 整数 | 输出图像数量，1-6，默认 1 |
| prompt_extend | 布尔 | 是否启用提示词扩展，默认关闭 |
| watermark | 布尔 | 是否添加水印，默认关闭 |
| seed | 整数 | 随机种子，范围 0-2147483647 |
| size | 下拉选择 | 输出图像尺寸，提供多种常用宽高比 |
| image_1~3 | 图像（可选） | 最多 3 张参考图像 |

### Qwen 可选尺寸

| 宽高比 | 可选分辨率 |
|---|---|
| 1:1 | 512×512, 768×768, 1024×1024, 1536×1536, 2048×2048 |
| 3:2 / 2:3 | 768×512, 1536×1024 / 512×768, 1024×1536 |
| 4:3 / 3:4 | 1024×768, 1536×1152 / 768×1024, 1152×1536 |
| 16:9 / 9:16 | 1024×576, 1536×864, 1920×1080 / 576×1024, 864×1536, 1080×1920 |

## 节点 2：Wan 2.6 Image Generation

路径：`Qwen` → `Wan 2.6 Image Generation`

### 两种模式

| 模式 | enable_interleave | 说明 |
|---|---|---|
| **图像编辑** | `false`（默认） | 基于 1~4 张输入图像进行编辑/风格迁移/主体一致性生成 |
| **图文混排** | `true` | 根据文本生成图文并茂的内容（0~1 张参考图） |

### 输入参数

| 参数 | 类型 | 说明 |
|---|---|---|
| text | 文本框 | 提示词 |
| enable_interleave | 布尔 | 模式切换：关闭=图像编辑，开启=图文混排 |
| size | 下拉选择 | 输出尺寸（支持 1K/2K 简写和具体分辨率） |
| n | 整数 | 输出图像数量 1-4（仅图像编辑模式） |
| max_images | 整数 | 最大图像数量 1-5（仅图文混排模式） |
| prompt_extend | 布尔 | 提示词智能改写（仅图像编辑模式），默认开启 |
| watermark | 布尔 | 是否添加水印，默认关闭 |
| seed | 整数 | 随机种子，范围 0-2147483647（0 表示随机） |
| image_1~4 | 图像（可选） | 编辑模式最多 4 张，混排模式最多 1 张 |

### Wan 可选尺寸

| 选项 | 实际值 | 说明 |
|---|---|---|
| 1K (跟随输入图比例) | `1K` | 总像素≈1280×1280，比例跟随最后一张输入图 |
| 2K (跟随输入图比例) | `2K` | 总像素≈2048×2048，比例跟随最后一张输入图 |
| 1:1 (1280×1280) | `1280*1280` | — |
| 2:3 (800×1200) | `800*1200` | — |
| 3:2 (1200×800) | `1200*800` | — |
| 3:4 (960×1280) | `960*1280` | — |
| 4:3 (1280×960) | `1280*960` | — |
| 9:16 (720×1280) | `720*1280` | — |
| 16:9 (1280×720) | `1280*720` | — |
| 21:9 (1344×576) | `1344*576` | — |

> 图文混排模式下 `1K`/`2K` 不可用，请选择具体分辨率。

## 输出

两个节点的输出格式相同：

| 输出 | 类型 | 说明 |
|---|---|---|
| images | IMAGE | 生成的图像（支持批量） |
| log | STRING | 详细日志：模型名称、提示词、输入/输出图像数量、URL、分辨率、格式、文件大小。Wan 图文混排模式还包含生成的文本内容。 |
